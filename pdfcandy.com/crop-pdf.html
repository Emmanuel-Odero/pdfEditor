How to crop PDF

Here’s the guide on how to crop a PDF with PDF Candy: add a file to the tool (use the “Add file” button or drag and drop the file into the browser window), move the vertical and horizontal dotted lines on the borders of the uploaded file preview to set the new borders for the document layout and press the “Crop PDF” button. Get the new file as soon as it is processed.

Easy file upload

You can upload the PDF files for processing not only from your device but from Google Drive and Dropbox accounts as well. The same is for the newly created files: they can be downloaded to the device or uploaded to Google Drive or Dropbox right away.

Cross platform PDF cropper

Crop PDF instantly from any OS you are using. Windows, Mac, Chromebook, Linux are supported as well as iOS and Android. Thus, you can crop PDF files not only on your PC or laptop but on the go on a tablet or smartphone as well.
